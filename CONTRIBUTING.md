# Contributing

## Configure

Windows:

```
nmake configure
```

Non-Windows:

```
make configure
```

## Build

Windows:

```
nmake build
```

Non-Windows:

```
make build
```

## Examples

Windows:

```
nmake example
```

Non-Windows:

```
make example
```

## Test

Windows:

```
nmake test
```

Non-Windows:

```
make test
```
